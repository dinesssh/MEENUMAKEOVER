# Meenu Makeover Studio — Luxury Web Application

## Tech Stack
- Next.js 16 (App Router)
- React 19
- Tailwind CSS v4
- GSAP for Animations
- Sanity CMS for Content Management
- Plausible for Privacy-First Analytics
- Vercel for Deployment

## Quick Start
```bash
npm install
npm run dev
```

## Features
- Infinite Marquee Testimonials
- Filterable Bridal Gallery (Sanity-backed)
- Luxury Typography System
- Exit Intent Modal & Floating CTAs
- Full Responsive & Mobile-First Design
